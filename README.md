# student.management..py
The Student Management System is a Python command-line application used to manage student records.
It allows users to add, update, delete, and view student information such as name, ID, and grade.
The  project uses Object-Oriented Programming(OOP) concepts with classes and stores data in a JSON or OS file, ensuring records are saved even after the program is closed.
It also validates unique student IDs provides a simple, user-friendly menu.
